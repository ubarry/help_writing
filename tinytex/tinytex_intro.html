1 使用 R 安装 TinyTeX

TinyTeX 的原生态是 R, 通过 R 安装 TinyTeX 最为方便；当然非 R 用户也可以安装 TinyTeX, 只不过相对更为麻烦。非 R 用户如何安装 TinyTeX, 参见 Section 5

首先安装 R 包 tinytex:

install.packages("tinytex")

然后利用 tinytex 包的 install_tinytex() 来安装 TinyTeX. 我们可以先加载该包：library("tinytex"), 然后使用 install_tinytex() 函数， 这样你不需要在函数名前加包的名字；我们也可以不加载包，直接通过这种方式使用该包的函数：pkg_name::fun_name()

若我们对该函数不熟悉，可以通过 ?tinytex::install_tinytex 来查看该函数的帮助。请关注该函数以下参数

  • force: 如果侦测到还存在其它 LaTeX 发行版，或 dir 参数指定的路径存在，force = FALSE （默认值）将取消安装，取值为真则强制安装
  • dir: TinyTeX 安装路径，注意该路径必须不存在，除非 force = TRUE
  • version: TinyTeX 的版本，如 “2022.12”. 默认安装 latest daily build of TinyTeX. 若 version = "latest", 则安装 latest monthly Github release of TinyTeX.
  • bundle:
  • repository: CTAN 仓库，默认为 https://mirror.ctan.org, 我们可以将仓库指定为境内镜像，如 http://mirrors.tuna.tsinghua.edu.cn/CTAN/, 路径余下部分 /systems/texlive/tlnet 将自动添加
  • add_path: 是否将 bin path of TeXLive 添加到环境变量，默认为真。我们也可以在终端 tlmgr path add 来添加。这样我们可以在终端直接使用 “xelatex”, “pdflatex” 来生成 pdf 文档

tinytex::uninstall_tinytex() 函数可以卸载安装的 TinyTeX

上述安装需要从 github 下载安装文件，由于众所周知的原因，安装过程往往会中断。替代方案是线下安装

步骤1：从 https://github.com/rstudio/tinytex-releases/releases 下载安装文件，如 “TinyTeX-v2022.12.zip”. 注意 “TinyTeX1”, “TinyTeX2” 和 “TinyTeX” 包含不同的 LaTeX 包，因此文件大小也不同

步骤2：在 R 中使用下述命令离线安装已下载的文件。注意包名 “tinytex” 和函数名 “install_prebuilt()” 之间有 3 个英文冒号，因为该函数是 tinytex 的 non-exported function

tinytex:::install_prebuilt(pkg = "e:/down/sftwr/TinyTeX-v2022.12.zip",
                           dir = "d:/tinytex")

这样，我们将 TinyTex 安装在 d:/tinytex 路径下。若我们不设置参数 “dir”, 则 TinyTex 将安装在默认的路径下，Windows 用户一般为“我的用户”下的 “AppData/Roaming/TinyTeX”, 我们可以使用函数 tinytex::tinytex_root() 得到该路径

2 在 R 中安装 LaTeX 包

列出已安装的包

tinytex::tl_pkgs()

这里, tl 指的是 “TeXLive”

设置 CTAN 镜像

tinytex::tlmgr_repo(url = "http://mirrors.tuna.tsinghua.edu.cn/CTAN/")

国内常用的其他镜像还有

  • aliyun
  • ustc
  • …

搜索包

tinytex::tlmgr_search("titling.sty")

安装包

tinytex::tlmgr_install("titling.sty")

tinytex:::install_yihui_pkgs() installs packages needed for building PDF vignettes of many CRAN packages

升级包

tinytex::tlmgr_update()

如果你看到如下错误信息 “Remote repository newer than local”, 这说明 TinyTeX 本身版本太旧了，你需要升级 TinyTeX 了

tinytex::reinstall_tinytex()

你也可以先卸载老版本的 TinyTeX, 然后安装新版本的 TinyTeX

3 尝试 LaTeX

假如文件 “e:/doc/test.tex” 包含如下内容

\documentclass{ctexart}
\title{Test File of TinyTex}
\author{Rho Zhang}
\date{}
\begin{document}
TinyTex is a light-weight, portable \LaTeX \ based on TeXLive, 
it was developed by Yihui, Xie

TinyTex 是一个轻量级、可移植的 \LaTeX \ 发行版，它基于 TeXLive, 
由谢益辉开发。

\end{document}

方法1: 在 R 中

tinytex::xelatex("e:/doc/test.tex")

方法 2: 在终端如 PowerShell

cd e:/doc
xelatex test.tex

缺少包的错误，解析错误日志

parse_packages("test.log")

安装 ctex 包

tlmgr_install("ctex")

4 终端相关命令

# 设置或修改 CTAN 镜像
tlmgr option repository https://mirrors.tuna.tsinghua.edu.cn/CTAN/systems/texlive/tlnet

# 搜索包
tlmgr search --global --file "unicode-math.sty"

# 安装包
tlmgr install unicode-math

# 罗列需要更新的包
tlmgr update --list

# 更新所有的包
tlmgr update --self --all

5 非 R 用户安装 TinyTeX

从 https://yihui.org/tinytex/install-bin-windows.bat 下载批处理文件，然后双击安装

上述安装需要 PowerShell (至少 3.0 版本)

移除 TinyTeX, 只需删除安装文件夹即可
